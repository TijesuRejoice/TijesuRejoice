# VSToFlowHelper

**VSToFlowHelper** is a lightweight Visual Studio Code extension that enables developers to **generate flowcharts from code snippets** in real-time using an integrated web-based visualization tool.

This extension streamlines the process of converting simple control flow structures—like `if/else`, `for` loops, and expression statements—into clean, readable Mermaid.js flowcharts. Ideal for **students**, **educators**, and **developers** looking to understand or explain logic visually.

## Features

- Send highlighted code from your editor to the companion web app
- No configuration or language setup required
- Supports **Python**, **JavaScript**, and **C++**
- Opens rendered flowchart in browser immediately

## Requirements

Ensure your default browser supports **ES6** and has **JavaScript Enabled**
